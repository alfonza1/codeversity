Stores and serves educational content

CreateCourse: /courses (POST) - Add a new course.
UpdateCourse: /courses/{id} (PUT) - Update course details.
DeleteCourse: /courses/{id} (DELETE) - Remove a course.
GetCourses: /courses (GET) - List available courses.
GetCourseDetails: /courses/{id} (GET) - Get detailed information about a course.

API Security

multiple modules make a course. each module needs a title, each module needs a description.
each course needs a title, list of subjects, one language, and one career. each course needs a description.

tests run before pushed. if any fails then the push is rejected.
